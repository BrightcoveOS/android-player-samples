# Offline Playback

These samples show how to download Brightcove Video Cloud content for offline playback and manage those downloads with the Brightcove offline-playback plugin.

## Requirements

- **Platform:** Android phone and tablet.
- **Minimum OS:** Android 5.0 (API 21).
- **Toolchain:** Android Studio, JDK 17, Gradle 8.13.
- **Extra SDKs:** the Brightcove offline-playback plugin, resolved from Maven and wired automatically by the build.

## Setup

The sample plays from the shared Brightcove demo account (`5420904993001`). Offline downloads require content licensed for download in Video Cloud.
