# Widevine DRM (WidevineModularSampleApp)

Plays Widevine Modular DRM-protected content; license handling is built into the SDK.

## Key files

| File | Responsibility |
|---|---|
| `MainActivity.kt` | Loads the Widevine-protected sample video from the Brightcove catalog and plays it. |
