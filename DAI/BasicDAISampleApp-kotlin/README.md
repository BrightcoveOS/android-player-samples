# DAI (BasicDAISampleApp)

Plays a stream with Google Dynamic Ad Insertion, where ads are stitched into the stream server-side.

## Key files

| File | Responsibility |
|---|---|
| `MainActivity.kt` | Configures the Google DAI component and plays the ad-stitched VOD stream for the sample video. |

See the [DAI README](../) for shared setup and requirements.
